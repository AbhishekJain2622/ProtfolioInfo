## 🚀 About the Project

Welcome to **Abhishek Jain's Personal Portfolio Website**!  
This website showcases:
- My featured **projects**.
- My **resume** and technical **skills**.
- A modern **responsive design** built with React.js.

🔗 **Live Website:** [abhishekjain.tech](https://abhishekjain.tech)

---

## 🛠 Built With

This project was built using the following technologies:
- ⚛️ **React.js** - Frontend Framework
- 🌐 **Node.js** + **Express.js** - Backend
- 🎨 **CSS3** - Styling
- 🖥️ **Visual Studio Code** - Development Environment
- 🚀 **Vercel** - Deployment

---

## ✨ Features

- **📖 Multi-Page Layout**: Organized and structured layout.  
- **🎨 Styled with React-Bootstrap and CSS**: Easy to customize colors and themes.  
- **📱 Fully Responsive**: Works seamlessly across all devices.
